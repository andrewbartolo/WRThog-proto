WRThog-proto
--------------
Written in node.js (prototype, see WRThog).  Router + internet appliance penetration testing tool.  Tests HTTP webadmin default logins across a wide range of devices.

Dependencies
--------------
  * node.js
    * commander
    * evilscan
    * geoip-lite
    * open
    * qjobs
    * request

Please use responsibly.
